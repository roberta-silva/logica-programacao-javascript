# Lógica de Programação e Algoritmos com JavaScript

Repositório contendo minha jornada de aprendizado através dos exercícios do livro "Lógica de Programação e Algoritmos com JavaScript" de Edécio Fernando Iepsen.



## Objetivo

Este repositório documenta meu progresso no estudo de lógica de programação e algoritmos, servindo como:

- Registro do meu aprendizado e evolução
- Portfólio de estudos em JavaScript
- Referência futura para conceitos fundamentais

## Como Executar

### Pré-requisitos

- [Node.js](https://nodejs.org/) v14 ou superior
- Navegador web moderno (Chrome, Firefox, Edge)

### Exercícios em Node.js

```bash
node caminho/do/arquivo/exercicio.js
```

### Exercícios no Navegador

Abra o arquivo `.html` correspondente diretamente no navegador, ou use a extensão **Live Server** do VS Code para uma experiência com recarregamento automático.
