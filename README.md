# Lógica de Programação e Algoritmos com JavaScript

Repositório contendo minha jornada de aprendizado através dos exercícios do livro **"Lógica de Programação e Algoritmos com JavaScript"** de Edécio Fernando Iepsen.

## Objetivo

Este repositório documenta meu progresso no estudo de lógica de programação e algoritmos, servindo como:
- Registro do meu aprendizado e evolução
- Portfólio de estudos em JavaScript
- Referência futura para conceitos fundamentais
- Material de consulta para outros estudantes

## 📖 Sobre o Livro

**Título:** Lógica de Programação e Algoritmos com JavaScript  
**Autor:** Edécio Fernando Iepsen  
**Editora:** Novatec

O livro aborda fundamentos essenciais de programação através da linguagem JavaScript, cobrindo desde conceitos básicos até estruturas de dados mais avançadas.

## Como Executar

### Pré-requisitos
- Node.js instalado (versão 14 ou superior)
- Navegador web moderno (Chrome, Firefox, Edge)

### Executando no Node.js
```bash
node caminho/do/arquivo/exercicio.js
```

### Executando no Navegador
Abra o arquivo HTML correspondente diretamente no navegador ou utilize a ferramenta Live Server do VS Code.
